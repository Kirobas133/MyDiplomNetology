Паук
Программа сделана на 2 пула потоков
1 пул загружает сайты, задачу для второго пула для работы с базой.
2 пул очищает строку от всего кроме слов, грузит базу.

Подробные комментарии в коде 
код разбит по пулам для удобства.

В базе две таблицы  
1 ссылки на сайты
2 слова и маркер на первую таблицу
